Methods

Methodology

The project aims to address the following questions:
RQ: If the life of an individual is full of meaningful experiences connected to times and places, how can text and sentiment analysis illustrate these events?
RQ: Does color describe and/or shape the memory of an experience?

Using an online survey, information concerning the participant’s demographics (age range, gender, location at the time of the event), thoughts, emotions, a description of events surrounding a day meaningful to them since the onset of worldwide quarantine procedures invoked by the COVID-19 pandemic and why, a color picker, and a description of why they selected that particular color.

Procedure

The project will include the following phases and procedures:

  1. Design. The data collection instrument,the online survey, will be designed and pilot tested.
  2. Analysis of COVID-19 case related data Data from the COVID-19 Data Repository by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University will be collected and analyzed.
  3. Participant Recruitment Adults, 18 years of age or older, will be recruited through institutional mailing lists and social media platforms (i.e., Twitter, Instagram, Reddit) over the course of two weeks.
  4. DataCollection Using an online survey, a confidential sample of 20-50 participants’ thoughts,emotions, and description of events surrounding a day meaningful to them since the onset of worldwide quarantine procedures invoked by the COVID-19 pandemic.
  5. Data Analysis Once cleaned of any and all identifying information, the data will be analyzed with Python. All code, cleaned data, and notebooks will be stored in a GitHub repository.
  6. Data Visualization Using the interactive Plotly Python Graphing Library, a visualization of the data and data dashboard will be incorporated into a GitHub Pages website constructed in HTML/CSS.
